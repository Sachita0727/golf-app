## ✨ Features

### 👤 User

* Signup / Login
* Subscription (Monthly / Yearly)
* Select Charity
* Add Golf Scores (Max 5)
* Participate in Draw
* View Results

### 🎲 Draw System

* Random number draw (1–45)
* Match system:

  * 5 → Jackpot
  * 4 → Great
  * 3 → Good

### ❤️ Charity

* User selects charity
* Contribution shown

### 🧑‍💻 Admin

* Run Draw
* View Results
